# Release Policy

Tsova Research Kit is intended for defensive documentation and responsible research.

## Allowed content

- Defensive security templates
- Learning notes
- Responsible disclosure drafts
- Sanitized examples
- Lab-only examples

## Prohibited content

- Stolen data
- Private credentials
- Exploit instructions against real targets
- Sensitive screenshots
- Unauthorized target details

## Public examples

Examples should be fictional, lab-based, or heavily sanitized.

## Approval

Any real-world disclosure should be reviewed before publication.
