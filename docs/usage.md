# Usage Guide

## Create a new case

From the project root:

python tools/create_case.py example-target

The tool creates a dated case folder under `cases/`.

## Recommended workflow

1. Confirm scope.
2. Create a case folder.
3. Fill out recon notes.
4. Save evidence carefully.
5. Draft findings.
6. Prepare disclosure email if needed.
7. Complete the retrospective.

## Safety notes

- Only test assets you are authorized to test.
- Do not publish sensitive details without approval.
- Avoid including secrets, tokens, private customer data, or exploit-ready instructions.
